# Hyper Max Store - Snacks and Chips Database System

BTEC Higher National Diploma in Computing - Software Engineering
Unit 4: Database Design & Development

## Overview
A relational database system for managing products, customers, suppliers, 
and sales for the Snacks and Chips department at Hyper Max Store.

## Technologies
- Microsoft SQL Server
- Visual Studio 2022 (C# Windows Forms .NET Framework)
- ADO.NET (System.Data.SqlClient)

## Database Tables
- Category
- Supplier
- Product
- Customer
- Basket
- BasketItem

## Features
- Full CRUD operations
- User role-based security (StoreManager, Cashier)
- Data validation with NOT NULL, CHECK, and FOREIGN KEY constraints
- SQL queries with JOIN across multiple tables
- Windows Forms user interface
